# Vibe Coding a Personal Site

Sometimes, the best way to learn modern web development is to throw away the templates and just build something that *feels* right. That was the ethos behind this portfolio site. I wanted a platform that merged enterprise-grade performance with an edgy, cyberpunk-inspired aesthetic. 

## The Architecture
I chose **Next.js** for its App Router, Server Components, and seamless static generation capabilities. 

### Why Next.js?
- **Speed**: By statically generating the pages (SSG) at build time, the site loads near-instantly on any connection.
- **Developer Experience**: The file-based routing and built-in CSS modules made it incredibly easy to iterate on designs quickly.
- **Markdown CMS**: I built a custom, local Markdown-based Content Management System using `gray-matter` and `remark`. This allows me to write blogs (like this one!) in simple text files without ever touching the React code again.

## The Aesthetic: Enterprise Meets Edgy
I wanted to break away from the generic, bright-white corporate portfolios. 
- **Dark Mode by Default**: Deep charcoal backgrounds reduce eye strain and make the accent colors pop.
- **Neon Accents**: Cyan and purple gradients provide a futuristic, tech-forward "vibe."
- **Glassmorphism**: Semi-transparent, blurred cards give the UI depth and texture.

## AI-Assisted "Vibe Coding"
To accelerate development, I utilized advanced AI coding assistants. This allowed me to act as the architect and creative director, dictating the "vibe" and business logic while the AI handled the boilerplate React components and CSS animations. It was a fascinating exercise in modern software engineering where human creativity pairs with AI velocity.
